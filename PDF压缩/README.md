## 一个简单的pdf压缩脚本

## Required
Python3


## Windows使用

1. 安装相关
[Ghostscript](https://ghostscript.com/releases/index.html)
注意：代码中的路径要更换为安装的地方。

2. 使用脚本
```
# 压缩单个PDF
python pdfc.py -f {file_name}
# 批量压缩PDF
python pdfc.py -r {folder_path}
```

2. 使用脚本
```
# 压缩单个PDF
python/python3 pdfc.py -f {file_name}
# 批量压缩PDF
python/python3 pdfc.py -r {folder_path}
```
